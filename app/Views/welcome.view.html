Seja Bem-vindo ao Nopake!

Nopake é um microframework escrito em PHP com foco em MVC. O Nopake utiliza uma sintaxe de template chamado VH para renderização da UI.

Documentação

Componentes

Use componentes para o reaproveitamento do código e a separação da interface de usuário das demais camadas.

MVC

O Nopake utiliza o padrão MVC para separar as camadas de controle, modelo e visual no desenvolvimento de sua aplicação.
